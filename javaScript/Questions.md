#### How to swap two values in JS without using the temp variable
```
a = 1, b = 2
Swap both a and b
Solution :
  [a,b] = [b,a]
```

obj = {"manoj" : [{color:"blue"},{color:"red"}]};
obj[manoj]

obj["manoj"]
